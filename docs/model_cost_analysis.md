### 关键发现

1. **Claude Opus 4.6 相比 4.5 (4.1) 降价 3x**: 单价从 $15/$75 降到 $5/$25，加上轮次更少（DSV1: 29 vs 82），每样本费从 $20.51 降到 $3.55 (DSV1)，$82.00 降到 $28.73 (DSV2)。
2. **费用主要由 input 驱动**: 多轮对话累积 context 是最大开销。轮次越多，累积 input 指数级增长。Claude Opus 4.5 的 82 轮平均轮次导致 input 爆炸。
3. **国产模型性价比极高**: qwen3-max ($0.26-$3.32/样本)、gemini ($0.53-$1.76)、kimi ($0.64-$2.41) 均在 $5 以内。
4. **GLM-4.7 成功率低**: DSV1 仅 4/14 成功（可能有 10 个样本失败/未跑），DSV2 仅 3/10。虽然单价便宜，但每个成功样本耗时极长（75-126min），效率最低。
5. **DSV2 比 DSV1 贵 3-10x**: 中长篇任务的轮次和 context 大幅增加。gemini 从 $0.53 到 $1.76 (3.3x)，claude-opus-4-6 从 $3.55 到 $28.73 (8.1x)。
