## 类型风格映射

| 类型 | 命名建议 | 示例 |
| --- | --- | --- |
| 武侠 | 复姓+古雅字，硬朗或清亮音韵 | 欧阳 + 清、慕容 + 峰 |
| 仙侠 | 清亮意象字，轻盈音韵 | 南宫 + 云、令狐 + 曦 |
| 古言/历史 | 单姓或复姓+温润/厚重字 | 谢 + 文、长孙 + 礼 |
| 都市/现实 | 常用单姓+现代中性字 | 王 + 宁、林 + 晨 |
| 商战/职场 | 简洁+明朗/硬朗字，易记 | 李 + 博、程 + 言 |
| 悬疑/犯罪 | 冷调/锐利音韵+中性字 | 周 + 影、韩 + 清 |
| 科幻 | 简洁+现代感字，跨语种友好 | 陈 + 宇、徐 + 维 |
| 言情/文艺 | 柔美/温润字+明晰音韵 | 宋 + 雅、吴 + 诗 |
