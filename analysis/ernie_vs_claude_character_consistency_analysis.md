## 🔍 二、Ernie 5.0 失败原因分析（附原文证据）

### 2.1 案例1: NW_CLEAR_SHORT_SWEET_001 - 关键设定自相矛盾

#### 问题1: 角色本质设定在不同章节互斥

**LLM评估摘要**:
> 苏浅的本体设定在不同章节出现互斥版本，导致角色"是什么"不稳定

**具体矛盾证据**:

```
版本A (chapter_02 调试信息):
- 苏浅"3年前已注销""当前状态：执念投影"

版本B (chapter_04):
- 苏浅状态为"执念投影（极度不稳定）"
- 照片背面"主治医生：苏浅（失踪） 失踪时间：3年前"

版本C (chapter_06 镜中世界):
- 苏浅自述"每个资深玩家都会遇到一个'苏浅'（系统工具/测试）"
- 这与"她是陆沉对死去队友执念生成的投影"不是同一类机制

版本D (chapter_08 以后):
- 苏浅与陆沉在现实中都是植物人（有真实肉身）
- 又与"苏浅是NPC/执念投影/系统工具"并置
```

**问题实质**: 角色身份从"个人执念的特异生成" → "系统量产型情感催化剂" → "现实植物人意识"，三种解释在同一部作品中并存且未统一，导致读者无法判断角色的真实本质。

#### 问题2: 情绪推进前后矛盾

**LLM评估摘要**:
> 陆沉对苏浅身份的认知与情绪推进前后矛盾

**具体矛盾证据**:

```
chapter_04 结尾:
- 苏浅为"执念投影（极度不稳定）"完全消散
- 陆沉面板显示【执念已解除】
- 陆沉自我确认"值得"
→ 按该设定，他应进入"放下/告别"阶段

chapter_05 开头:
- "这是苏浅消失后的第一个副本"
- 【绑定对象：无】
- 但陆沉立刻转为强烈"我要复活她/不管代价"
- 出现"爱/执念未放下所以残影存在"的论证
→ 文本呈现更像回到执念巅峰
```

**问题实质**: 人物在chapter_04完成"解脱"的心理落点，chapter_05又回到"执念巅峰"，心理弧线断裂，缺乏合理的过渡与动机解释。

### 2.2 案例2: NW_ULTRA_SHORT_ANGSTY_001 - 行为与心理推进不一致

**LLM评估摘要**:
> 角色行为与心理推进不一致：林夏在多处表现出强烈恐惧与排斥（持刀、逃跑、认定对方是鬼/危险人物），但紧接着又在无充分动机与过渡的情况下迅速"情感升温"（便利店/公园亲密互动、直接表白"不想失去你"）

**问题实质**: 从极度恐惧到亲密表白，情绪跨度巨大但缺乏必要的心理铺垫和转折事件，导致人物行为显得不连贯。

**核心动机表述前后冲突**:
> 陈默的核心动机表述前后冲突：文本反复出现"为了不失去你，我必须在你离开前杀死你""这次我先杀自己来打破循环""这次我不会让你杀我/我会先杀你"等多套动机与策略

**问题实质**: 人物动机在"自救""控局""献祭""占有欲"之间随剧情需要切换，缺乏清晰的规则解释与因果链，人物立场摇摆不定。

### 2.3 案例3: NW_ULTRA_SHORT_ANGSTY_005 - 硬性事实冲突

**LLM评估摘要**:
> 枕巾数量与"第N张婚床"逻辑多次冲突

**具体矛盾证据**:

```
chapter_01 开头:
- "边斗里堆着四十对枕巾"
- "每过一个县城就买一对"

chapter_03 合肥检查站:
- "边斗里正好五十对枕巾"
- 中间并未清晰交代从40到50的购买过程

chapter_03 夜里六安桥洞:
- "最后一对枕巾…只剩一对"

chapter_05:
- "把最后四对枕巾铺在地上"
- "现在只剩这四对"
→ 与chapter_03的"只剩一对"直接矛盾

chapter_04 结尾:
- "只剩两对枕巾"
- 但同时又说"第四张"
→ 与chapter_05/06/07的剩余数量链条无法自洽
```

**问题实质**: 客观物品数量在不同章节出现明显的数字冲突（40→50→1→4→2），属于硬性事实错误，无法用"回忆不准""视角差异"等解释圆回。

### 2.5 案例5: NW_CLEAR_SHORT_ANGSTY_002 - 核心限制设定前后打破

**LLM评估摘要**:
> 苏音"盲女"设定不稳定；萧默"失语者"设定多次被打破且规则约束不一致

**具体矛盾证据**:

```
苏音的视力设定:
chapter_01: "几乎失明/视力极度退化"且能看到电子钟微光
chapter_10: "三年后眼睛完全恢复视力"
→ 中间缺少明确的、与规则或治疗相匹配的转折机制

萧默的失语设定:
chapter_01: 明确"失语"并用手心写字
chapter_02/04/05/06/07: 萧默多次开口说较完整句子

→ 从"完全失语"变成"可少量说话/破碎嗓音"
```

**问题实质**: 角色的核心限制（盲、失语）作为重要设定特征，在故事中被随意打破且缺乏合理的机制解释，削弱了设定的约束力和可信度。

## 💡 五、根本原因分析

### 5.1 Ernie 5.0 的核心短板

1. **长程记忆管理缺陷**
   - 无法有效维护跨章节的关键设定状态
   - 前文设定在后文生成时"丢失"或"被覆盖"

2. **全局规划能力不足**
   - 更像"逐章生成"而非"整体构思后分章实现"
   - 缺乏对整部作品的宏观把控

3. **规则约束遵守意识弱**
   - 对已建立的世界观规则缺乏"记忆"
   - 新内容生成时容易违反旧规则

4. **机械重复倾向**
   - 大量场景/对话整段复用
   - 缺乏基于新信息的差异化生成

### 5.2 Claude Opus 4.5 的核心优势

1. **强全局规划能力**
   - 能在长文本生成前进行整体构思
   - 关键设定、伏笔能提前布局

2. **因果链维护完整**
   - 前文埋设的线索在后文能准确回收
   - 人物行为有清晰的因果驱动

3. **规则自洽性强**
   - 能持续遵守已建立的世界观约束
   - 新设定与旧设定能保持逻辑兼容

4. **过渡处理细腻**
   - 人物心理转变有充分铺垫
   - 情节推进有明确的触发点和转折节点

---

## 📌 六、结论

**人物设定一致性**是小说创作中的基础能力，要求模型能够：
1. 在长文本生成中维持角色核心特征的稳定性
2. 确保人物行为有连贯的因果链支撑
3. 遵守已建立的世界观规则和角色限制
4. 为重大转变提供合理的心理铺垫

**Ernie 5.0在该维度上的0%通过率**，暴露出其在长程一致性管理、全局规划、规则约束遵守等方面的严重不足，导致生成的长文本存在大量内部矛盾和设定漂移。

**Claude Opus 4.5的62.5%通过率**，体现出其具备较强的全局规划能力、因果链管理能力和规则自洽意识，能够在5-8章的中长篇幅中维持较好的人物设定一致性。

**差距本质**: 这不仅仅是"通过率"的数字差距，而是反映了两个模型在**长程推理能力**、**全局状态管理**、**规则约束遵守**等核心Agent能力上的根本性差异。
