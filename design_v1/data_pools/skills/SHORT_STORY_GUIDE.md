# 短篇故事创作指南 (6500-10000字)

**适用场景**：用户明确要求短篇创作（6000字左右、1万字以内），适合番茄平台等短故事发布需求。

**核心特征**：单视角锁死，实时叙事，物件回扣，情绪单频。

---

## 写作建议

### 五个建议原则

1. **时间压缩**：避免大跨度时间跳跃（少用"三年后"，多用"三分钟后"）
2. **视角统一**：建议单主角视角，避免频繁切换视角
3. **角色控制**：建议第二幕后不引入新的重要角色
4. **闪回克制**：闪回场景建议<200字，且通过感官触发
5. **结尾凝练**：建议最后一句用名词或动词结尾，增强画面感

### 三幕结构建议（6500字）

- **第一幕 (0-1500字)**：快速切入，建立倒计时/紧迫感，完成第一次关键接触
- **第二幕 (1500-5000字)**：冲突螺旋升级（外部压制 → 内部崩解 → 不可逆行为）
- **第三幕 (5000-6500字)**：极限二选一，新稳态定格，物件回扣封蜡

### X-Y工具箱参考

**X轴建议选择**（单一核心冲突）：

- 牺牲所爱/爱上敌人/错误判断/复得失去
- 上下级竞争/为激情牺牲/亲族竞争
- 身份揭秘/疯狂/与神冲突/获取

**Y轴建议组合**（1-2个标签）：

- 追妻火葬场/读心术/重生/疯批强制爱
- 规则怪谈/身份错位/封闭空间/先婚后爱

**化学反应参考**：

- ↗ 甜爽（掌控变温柔）
- ↘ 虐心（献祭不可逆）
- ✷ 烧脑（认知颠覆）

### 七个写作技巧建议

1. **信息差设计**：读者知70%，主角知30%，保持悬念
2. **物件变形**：3个核心物件，每幕改变一次状态（完整 → 破裂 → 重组）
3. **实时感营造**：故事时间 ≈ 阅读时间（约30分钟），可量化时间流逝（电量87% → 63%）
4. **情绪聚焦**：建议1种主导情绪（如恐惧），所有变奏都是它的和弦
5. **身体接触**：每500字建议有非功能性身体接触（标记情绪，非推动情节）
6. **对话复调**：每句台词承载多重功能（推进情节+暴露关系+暗示秘密），善用未完成句（"我以为你——"）
7. **封蜡结尾**：物件回扣/感官余震/动作冻结（刀将刺未刺）

---

## 创作流程建议

1. **明确概念**：一句话故事核心 + 情感倾向 + 封闭场景
2. **配方诊断**：选定X模式 + Y标签 + 化学反应 + 设计3个核心物件
3. **分幕创作**：按三幕结构完成6500字完整文本
4. **自检要点**：
   - 字数是否在6500-10000之间？
   - 是否保持单一视角？
   - 3个核心物件是否都有回扣？
   - 最后一句是否以名词/动词结尾？
